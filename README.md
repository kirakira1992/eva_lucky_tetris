治愈风俄罗斯方块小游戏：奶油色 UI、消行狗狗特效、Eva 主题文案。纯原生 HTML + CSS + JS，无框架。

## 本地运行

双击根目录下的 `index.html` 用浏览器打开即可，或部署到任意静态托管。部署说明见 [DEPLOY.md](DEPLOY.md)。

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↓ | 软降 |
| 空格 | 硬降 |
| ↑ / X | 旋转 |
| C | Hold |
| P | 暂停 |
| R | 重新开始 |

# EVA Lucky Tetris 🐶

**English** | [中文](#)

A cozy, healing-style Tetris game: cream-colored UI, line-clear dog animation, and Eva-themed copy. Built with vanilla HTML, CSS, and JS—no frameworks.

---

## 🎮 Play Online

**Play in your browser:** [https://kirakira1992.github.io/eva_lucky_tetris/](https://kirakira1992.github.io/eva_lucky_tetris/)

---

## Run Locally

Open the project root and double-click `index.html` to play in your browser, or deploy to any static host. See [DEPLOY.md](DEPLOY.md) for deployment notes.

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left / right |
| ↓ | Soft drop |
| Space | Hard drop |
| ↑ / X | Rotate |
| C | Hold |
| P | Pause |
| R | Restart |
